# Uniformly

A web-based marketplace that enables parents to buy and sell pre-owned school uniforms within their communities. Uniformly promotes affordability, sustainability, and convenience by providing a dedicated platform for second-hand school uniform trading.

## Overview

Uniformly was developed to address the growing cost of school uniforms by creating a centralized platform where parents can list, browse, purchase, and sell pre-owned uniforms. The system offers secure user authentication, product management, payment integration, and automated email communication to ensure a seamless user experience.

## Features

* User registration and login system
* Secure session and cookie management
* Browse and search available uniforms
* Create, edit, and manage uniform listings
* Upload product images
* Shopping cart functionality
* Secure checkout process
* PayFast Sandbox payment gateway integration
* Email notifications for account and transaction activities
* User profile management
* Responsive design for desktop and mobile devices
* MySQL database integration for data storage

## System Architecture

The application follows a client-server architecture:

1. Users interact with the website through a web browser.
2. PHP processes business logic and user requests.
3. MySQL stores user accounts, listings, transactions, and platform data.
4. PayFast Sandbox handles payment processing during testing.
5. Email services send notifications and confirmations.
6. Cookies and sessions maintain user authentication and preferences.
